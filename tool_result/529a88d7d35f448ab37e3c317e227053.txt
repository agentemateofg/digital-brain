## 🛡️ Reglas para Evitar Bucle Infinito

**Objetivo:** Prevenir errores como `write_file() missing 2 required positional arguments`

### REGLA 1: SIEMPRE PROPORCIONAR ARGUMENTOS REQUERIDOS ⭐⭐⭐

```python
# ❌ MAL: write_file() sin argumentos
write_file()

# ✅ BIEN: write_file con todos los argumentos
write_file(file_path="/path/to/file.md", content="contenido")
```

### REGLA 2: VERIFICAR ANTES DE ESCRIBIR ⭐⭐

```python
# ✅ BUENO: Verificar si archivo existe antes de escribir
if not file_exists("/path/to/file.md"):
    write_file(file_path="/path/to/file.md", content="contenido inicial")
else:
    edit_file(file_path="/path/to/file.md", old_text="texto_antiguo", new_text="texto_nuevo")
```

### REGLA 3: USAR EDIT_FILE PARA ACTUALIZACIONES PEQUEÑAS ⭐⭐

```python
# ✅ BUENO: Para cambios pequeños, usar edit_file
edit_file(
    file_path="/Users/piter/.copaw/MEMORY.md",
    old_text="texto_antiguo",
```
